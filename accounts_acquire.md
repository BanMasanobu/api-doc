# POST /api/v1/accounts/:id/acquire
金融機関のデータ再取得を行う

## Resrouce URL
https://moneyforward.com/api/v1/accounts/:id/acquire

## Parameters
name | Description 
-----------|------------------------
id <br> *required* | 登録金融機関ID *hashed*

## Example

### request

> **POST** https://moneyforward.com/api/v1/accounts/e2RPphY9_-bdYZ1PBLMmjg==/acquire

### response-body

nothing
